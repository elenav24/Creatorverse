# Creatorverse - Elena Marquez's Pre-Work Submission for Codepath

This platform allows users to explore and manage a curated list of digital content creators.

### Completed Features
- Featured Creator Showcase: Displays a grid of 5 hand-picked creators with profile images and bios.

- External Linking: Direct "View their channel or page" links for every creator profile.

- Creator Directory: A dedicated View All Creators page to browse the full database.

- Content Contribution: Functional Add a Creator interface for user submissions.

- Responsive UI: A modern, mobile-friendly grid layout with a purple-themed aesthetic.
